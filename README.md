# Pacman Game

Welcome to the Pacman Game! This project is a web-based implementation of the classic Pacman game. It utilizes HTML, CSS, and JavaScript to create an interactive gaming experience.

## Features

- **Classic Gameplay**: Experience the traditional Pacman gameplay with ghosts and pellets.
- **Responsive Design**: The game is designed to fit within a specified area of the webpage.
- **Custom Font**: Unique font styling for the game title.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Modernizr (for feature detection)
